Extra Stuff

Your laptop must be running the last updated release of Windows 7 with 2GB RAM minimum or the most recent update of Windows 10 with 4GB RAM minimum. to use any of the files below.

Software

    * CHIRP - free, open-source tool for programming your amateur radio
    * Fldigi - free open-source program which allows sound card to be used as a data modem (PSK31, RTTY, etc)
    * Ham Radio Deluxe 5.24 - Final Free Version for rig control, digital, other stuff
    * N3FJP ARRL Field Day Contest Log - Logging software used for Field Day
    * PuTTY (32bit) or (64bit)- free SSH and Telnet client
    * WSJT-X (32bit) or (64bit) - open-source program designed for weak-signal digital communication (FT4, FT8, etc)
