Setting up Laptop for Logging

*** STOP *** IS YOUR LAPTOP UP TO DATE ???

Your laptop must be running the last updated release of Windows 7 with 2GB RAM minimum or the most recent update of Windows 10 with 4GB RAM minimum. The Field Day Network has NO Internet access.

Removing Last Years Field Day Software

N3FJP logging software is constantly updated. Everyone needs to be running the latest version. To remove previous installs of the N3FJP ARRL Field Day Contest Log Software perform the following:

    * Go to Control Panel.
    * Select Add or Remove Programs.
    * Click on ARRL Field Day Contest Log and select Change/Remove.
    * Pop-up windows will ask Are you sure, click Yes.
    * To be sure it is completely removed, run Windows Explorer and delete the following folder structure if they exist:

      C:\Program Files\N3FJP Software - Affirmatech\ARRL Field Day Contest Log
      C:\Program Files (x86)\N3FJP Software - Affirmatech\ARRL Field Day Contest Log
      C:\Users\(your ID)\Documents\Affirmatech

At this point your laptop is current and ready for a fresh install of the N3FJP ARRL Field Day Contest Log Software.

Installing N3FJP ARRL FD Logging Software

With the laptop ready for software installation, perform the following

    * Download the software from this server
    * Run the program and accept default choices. Click Finish when done.
    * When prompted to register, use W2ABCD and password 012345

      Note this license is for W2ABCD use only. Cannot be used for Home installations.

    * When the "Common Fields" window pops up, enter the following;

      Call: W2ABCD
      Country USA
      Latitude: 41.464
      Longitude: 74.014
      CQ Zone: 5
      ITU Zone: 8
      Class: 5A
      ARRL Section: ENY
      Operator: Enter your Call Sign
      Initials: Enter your Call Sign

5. Click DONE then quit the program.

Connecting Logging Software to the Server

    * From the Start menu run the following \\192.168.73.100\w2ho
    * A window explorer should pop-up showing two files; FD2021Log.mdb and read-write-test.txt
    * Open read-write-test.txt
    * Make a change to the text and save

      If there were no errors reported in any of the above steps, you are good to go for the next steps.

Using the Logging software

    * Start the Field Day software
    * When the Common Fields window pops-up, verify it is still populated with the information from the installation steps.
    * Click done to close the window.
    * Click File then Open
    * In the File name window that pops-up enter the following \\192.168.73.100\w2ho\FD2021log.mdb
    * When you are challenge for ID and password enter w2ho and ocarc-ny respectively.


You should now be connected. If you connected during Field Day operations you will see the Recent Contacts window fill up with contacts made other stations on site.
